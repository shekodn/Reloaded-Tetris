Tetris

El usuario nos esta pidiendo que habilitemos las opciones de guardar y cargar y ademas usemos música de fondo y un sonido para cada vez que encaja una pieza se escuche y que se animen las piezas mientras caen, el comentaba que le gustaria ver como si se resplandecieran o algo asi.

La opcion de Cargar sera con una C y la de Guardar con una G, ambas deberán de aparecer del lado derecho.

Criterio

20 Estandar de codificación

20 Animación de las piezas

30 Opción de Guardado funciona perfecto

30 Opcion de Cargado funciona perfecto
